About

JCAT is the software platform for the Market Design Competition (CAT) under the umbrella of the Trading Agent Competition (TAC). The CAT Competition supports multiple markets that run in parallel and compete against each other to attract traders and make profit. Each entrant of the CAT Competition runs a market, and set the market rules so as to win the game according to a certain set of assessment criteria. Traders, which are provided by the competition organizers, can move between markets to explore and select the best market to trade in to maximize their own profits. JCAT includes the CAT game server, trading strategies and market selection strategies to configure trading agents, i.e., traders, and a parameterized, reference implementation of market mechanism that enables entrants to get started easily.

JCAT has also been used extensively in experimental mechanism design research and education. If you would like to collaborate on JCAT-based research, or support further development of JCAT, please feel free to contact Jinzhong Niu.

Thank you for using JCAT!

Features:

  • High performance

  • Integration with ECJ for performing experiments involving evolutionary computing or genetic programming.

  • Highly extensible architecture

  • Highly configurable - experiments can easily be configured from parameter files

  • Pluggable pseudo-random number generators using the CERN colt library

  • Implementations of existing agents, learning algorithms, experiments and auction types described in the computational economics literature.

    • Development Status

    Acknowledgements

    This work is partly supported by EPSRC grant GR/T10671/01 - "Market Based Control of Complex Computational Systems." and NSF grant number IIS-9820657 - "Tools and Techniques for Automated Mechanism Design".

    The system is based on the 4-heap algorithm, described in the paper

    "Flexible Double Auctions for Electronic Commerce: Theory and Implementation" by Wurman, Walsh and Wellman 1998.

    JCAT makes use of the Mersenne Twister PRNG. Full details of the Mersenne Twister algorithm can be found in:

    Makato Matsumoto and Takuji Nishimura, "Mersenne Twister: A 623-Dimensionally Equidistributed Uniform Pseudo-Random Number Generator", in ACM Transactions on modeling and Computer Simulation, Vol. 8, No. 1, January 1998, pp 3--30.
